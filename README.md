# EStaffFe

## Installation frontend

```bash
$ npm i
$ npm run start
```

## Full installation
1. Run in terminal:
    ```bash
    $ mkdir e-staff && cd e-staff
    $ git clone https://github.com/defint/e-staff e-staff-be
    $ git clone https://github.com/defint/e-staff-fe
    $ cd e-staff-be
    $ docker-compose up
    $ cd ../e-staff-fe
    $ npm i
    $ npm run start
    ```
1. Go to [localhost](http://localhost:4200/)
